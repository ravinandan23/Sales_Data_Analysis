# Sales_Data_Analysis
Sales Data Analysis using Python (Pandas, Matplotlib)

This project analyzes retail sales data to identify sales trends, profit distribution, and regional performance.

Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

Key Analysis
- Sales by Region
- Profit by Category
- Quantity by Category
- Top States by Sales
- Sales vs Profit Relationship

Insights
- West region generated the highest sales
- Technology category produced the most profit
- High sales do not always guarantee high profit
